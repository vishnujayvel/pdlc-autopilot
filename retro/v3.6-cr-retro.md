## Root Cause: Prompt Blind Spots + No Markdown Linting

The PDLC process had two systemic gaps exposed by this self-upgrade:

1. **No markdown linting** — 3 of 17 issues (18%) were pure formatting (MD040, MD022, MD031) that a linter would auto-catch. The Actor's Linter/Type-Checker Gate only ran code linters.
2. **SKEPTIC lacks cross-reference verification** — 7 of 17 issues (41%) were internal inconsistencies: wrong paths, wrong counts, stale versions, missing phases. SKEPTIC verified code against spec but not content against itself.
3. **No placeholder detection** — `[PASTE ...]` literals survived into shipped artifacts because no validator scans for unresolved template placeholders.
